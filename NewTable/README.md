# Таблица с редактируемыми полями

Для таблицы передается объект данных из двух массивов:

```javascript
{
  DataMap: [ // В DataMap содержится описание полей таблицы (карта данных)
      {
        text: 'ID', // текст в заголовке
        align: 'start', // необязательный, выравнивание
        sortable: true, // необязательный, возможность сортировки по этому полю
        value: 'id', // имя поля в данных таблицы
        type: 'id', // необязательный, внутренняя настройка, тип поля, возможные варианты - image, title, text, string
      },
      {
        text: 'Title', // текст в заголовке
        sortable: true, // необязательный, возможность сортировки по этому полю
        value: 'title', // имя поля в данных таблицы
        type: 'string', // необязательный, внутренняя настройка, тип поля, возможные варианты - image, title, text, string
      },
  ...
  ],

  Data: [ // В Data массив с данными таблицы
      {
        id: 1, // имя поля id соответсвует описанному в DataMap[0].value === 'id'
        title: "Title 1", // имя поля title соответсвует описанному в DataMap[1].value === 'title'
        date: "2020-10-01",
        body: "Adding.... "
      },
      {
        id: 2,
        title: "Title 2",
        date: "2020-10-03",
        body: "the... ready... "
      },
  ...
  ]
}
```
